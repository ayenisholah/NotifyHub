**Project description (572/600 characters)**

Designed and built a production-shaped notification platform that turns one authenticated event into reliable email, SMS, and real-time in-app delivery. Implemented idempotent ingestion, PostgreSQL persistence, Redis/BullMQ workers, user preferences, quiet hours, digests, retries, a dead-letter queue, an embeddable React inbox, and an observability dashboard. Containerized and deployed it with Docker Compose, Nginx, and GitHub Actions. A controlled 10,000-event test sustained 199.96 notifications/second at p95 62.51 ms with zero HTTP failures or residual queue work.
